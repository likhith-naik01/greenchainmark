# CropAdvisor AI - Requirements

## System Requirements

### Node.js & Package Manager
- Node.js >= 18.0.0
- npm >= 8.0.0 (comes with Node.js)
- Git (for cloning repository)

### Database
- PostgreSQL >= 13.0

### Browser
- Modern browser with Web3 support (Chrome, Firefox, Edge)
- MetaMask browser extension

## Installation Commands

### Frontend Setup
```bash
npm install
```

### Backend Setup
```bash
cd server
npm install
```

### Global Tools (Optional)
```bash
npm install -g truffle          # Smart contract development
npm install -g ganache-cli      # Local blockchain for testing
```

## Environment Variables Required

### Frontend (.env)
```
VITE_CONTRACT_ADDRESS=          # Deployed smart contract address
VITE_NETWORK_ID=11155111        # Ethereum network ID (Sepolia)
VITE_API_URL=http://localhost:3001  # Backend API URL
```

### Backend (server/.env)
```
DB_USER=postgres                # Database username
DB_HOST=localhost               # Database host
DB_NAME=crop_advisor            # Database name
DB_PASSWORD=                    # Database password
DB_PORT=5432                    # Database port
GEMINI_API_KEY=                 # Google Gemini API key
PORT=3001                       # Server port
NODE_ENV=development            # Environment mode
CONTRACT_ADDRESS=               # Smart contract address
PRIVATE_KEY=                    # Ethereum private key (for deployment)
INFURA_PROJECT_ID=              # Infura project ID
```

- Google AI Studio API

## Security Considerations

### API Keys
- Store all API keys in environment files
- Never commit API keys to version control
- Use different keys for development and production

### Database
- Use strong passwords
- Enable SSL connections in production
- Regular backups recommended

### Smart Contracts
- Test thoroughly on testnet before mainnet deployment
- Use hardware wallets for production deployments
- Monitor contract interactions and gas usage
